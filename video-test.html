Cloudinary 기반의 UI 없는 순수 영상 배경입니다.

Cloudinary의 장점

  • 완벽한 UI 제거: 브라우저 기본 비디오 태그를 사용하여 유튜브 로고 등이 전혀 보이지 않습니다.
  • 자동 최적화 (f_auto): 접속한 브라우저에 맞춰 WebM, MP4 중 가장 효율적인 포맷을 전송합니다.
  • 성능 증대: 전용 CDN(Content Delivery Network)을 통해 영상 로딩 속도가 매우 빠릅니다.
  • 깔끔한 루프: 영상 끝부분에서 끊김 없이 자연스럽게 반복됩니다.
